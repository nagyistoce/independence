To activate your phone:

  1. Download any of the following iPhone firmware files from Apple:
    • Firmware version 1.0
    • Firmware version 1.0.1
    • Firmware version 1.0.2
    It doesn't matter which one, they all work equally well. I usually choose the one which matches my existing firmware.
  2. If your web browser didn't automatically decompress it to a folder, then change the file extension from .ipsw to .zip and unzip it yourself.
  3. Launch iNdependence.
  4. Select the Activation tab and press the Activate Phone button.
  5. You'll be prompted to choose the directory where you unzipped the firmware files.
  6. If you get an error message saying Error entering recovery mode then quit iNdependence, restart it, and try again.
  7. Wait for the success message. Note that the activation process will also enable YouTube for you.

Note that activation will leave your phone jailbroken. If you want to return your phone to jail, use the Return to Jail button in the Jailbreak tab.

To jailbreak your phone:

  1. Download any of the following iPhone firmware files from Apple:
    • Firmware version 1.0
    • Firmware version 1.0.1
    • Firmware version 1.0.2
    It doesn't matter which one, they all work equally well. I usually choose the one which matches my existing firmware.
  2. If your web browser didn't automatically decompress it to a folder, then change the file extension from .ipsw to .zip and unzip it yourself.
  3. Launch iNdependence.
  4. Select the Jailbreak tab and press the Perform Jailbreak button.
  5. You'll be prompted to choose the directory where you unzipped the firmware files.
  6. If you get an error message saying Error entering recovery mode then quit iNdependence, restart it, and try again.
  7. Wait for the success message.

To upgrade to 1.1.1 or 1.1.2 so that you can activate/jailbreak your phone:

Please note that if you have used anySIM 1.0 or other older free SIM unlocking solutions (anything except anySIM 1.1/1.2, gunlock, and iPhoneSimFree) to SIM unlock your phone, and you upgrade to 1.1.x, it will wipe out the SIM unlock and render the phone and text message part of your phone inoperable. Everything else will work though.

(Update)You only need to do this if you are upgrading to firmware 1.1.1 or 1.1.2. If you are upgrading to 1.1.3 or 1.1.4, then you can skip all of this directly upgrade via iTunes.

  1. Check the firmware version that's reported in the status line at the bottom of the iNdependence window and ensure that you are using firmware 1.0.2 (or lower).
  2. Activate your phone using the activation instructions above. Note that you should leave your phone jailbroken after activation.
  3. Install SSH on your phone using the instructions below.
  4. Ensure that your phone is connected to a Wi-Fi network (read the iPhone manual for instructions on how to set up Wi-Fi). Also ensure that it's IP address does not conflict with any other devices on your Wi-Fi network (sorry, I can't offer network admin help).
  5. Get the IP address for your phone by launching the Settings application on the phone, navigating to Wi-Fi and pressing the blue arrow beside the name of the Wi-Fi network that your phone is connected to.
  6. Select the Firmware tab and press the Pre Firmware Upgrade button.
  7. Enter the IP address of the phone and the root password (see below for the default root password if you didn't change it when you installed SSH).
  8. Follow the instructions given in the dialog windows. If iNdependence gives you a warning about host verification failure and asks to edit your known_hosts file, choose the Yes option (which is now the default).
  9. Once you are told to upgrade using iTunes, quit iNdependence, and hold down the option key while pressing the Update button in the iTunes window (not the Restore button). Then select the 1.1.1 or 1.1.2 firmware file when prompted.
  10. Wait for iTunes to finish.
  11. Quit iTunes and relaunch iNdependence. In the status bar at the bottom of the iNdependence window you should see Firmware 1.1.1 or Firmware 1.1.2 (depending on which one you upgraded to).
  12. Congratulations! You are now ready to activate/jailbreak your phone using the 1.1.x instructions.

Installing and Configuring SSH

Installing SSH is a breeze once your phone is jailbroken. Just choose the SSH tab and press the Install SSH/SCP/SFTP button. Then follow the instructions.

The default accounts on the phone are:

username: root
password: dottie

username: mobile
password: alpine

Remember to change the default passwords after installing SSH using the Change Password button. I'm not responsible if someone hacks your phone.

If you want to later remove SSH, you can do so using the Remove SSH button. iNdependence will ask you if you want to also remove certain files which might be shared with other applications on your phone at that time. Please read the description of those files carefully before you choose to remove them.

SIM Unlocking

SIM unlocking allows you to use your phone with any digital cellular service provider (as opposed to being tied to one provider and a long contract).

To perform a SIM unlock, simply select the Activation / SIM tab and press the SIM Unlock Phone button. It'll take a couple of minutes, so please be patient.

Customization

The Customize tab uses a Finder-style interface and allows for drag and drop installation of ringtones, wallpapers, and applications. You can also use the Add and Delete buttons.

Third Party Applications

Please note that in order to install/remove applications, you'll need to have SSH installed on the phone as you'll be asked for the phone's IP address and root password.

To find the phone's IP address, launch the Settings application, navigate to Wi-Fi and press the blue arrow beside the name of the Wi-Fi network that your phone is connected to.

Once SSH is installed, you can install third party applications by either dragging and dropping the applications from Finder into the System/User folder, or you can press the Add button to select the application from a file chooser. To remove an application, simply select it and press the Delete button.

Ringtones

Ringtones must be MPEG-4 audio (.m4a) files as the file chooser will only allow you to choose .m4a files.

Wallpapers

Wallpapers must be 2 PNG (.png) files (the main image and the thumbnail image). The wallpaper image sizes I use are 320x480 for the main image and 75x75 for the thumbnail image. These seem to work well for me.
