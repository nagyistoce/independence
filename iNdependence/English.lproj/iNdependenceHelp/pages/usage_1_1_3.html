iNdependence works well with firmware 1.1.3 and 1.1.4. You don't need to do any special steps when upgrading. Just upgrade as usual using iTunes and then activate/jailbreak/SIM unlock easily using iNdependence.

One thing to note with the 1.1.3/1.1.4 firmware is that the very first time you jailbreak/activate/SIM unlock your phone, iNdependence will give a message saying that it's generating a RAM disk file. This is normal. It takes a bit of extra time, but it will only be done once.

To activate your phone:

  1. Select the Activation tab and press the Activate Phone button. On occasion, iNdependence will give an error message saying that it failed to enter recovery mode. If this happens, just quit and restart iNdependence and try it again. Also, if you are waiting a long time and nothing changes on the phone screen, force quit and restart iNdependence and try again.
  2. You should see a message saying Waiting for jail break.... Please be patient as it will take a while to do this. At one point you'll see a bunch of text scrolling down the screen. This is normal.
  3. Wait for the success message. Note that the activation process will also enable YouTube for you.

To jailbreak your phone:

Note that this will be done automatically if you activate your phone.

  1. Select the Jailbreak tab and press the Perform Jailbreak button. On occasion, iNdependence will give an error message saying that it failed to enter recovery mode. If this happens, just quit and restart iNdependence and try it again. Also, if you are waiting a long time and nothing changes on the phone screen, force quit and restart iNdependence and try again.
  2. You should see a message saying Waiting for jail break.... Please be patient as it will take a while to do this. At one point you'll see a bunch of text scrolling down the screen. This is normal.
  3. Wait for the success message.

Installing and Configuring SSH

Installing SSH is a breeze once your phone is jailbroken. Just choose the SSH tab and press the Install SSH/SCP/SFTP button. Then follow the instructions.

The default accounts on the phone are:

username: root
password: dottie

username: mobile
password: alpine

Remember to change the default passwords after installing SSH using the Change Password button. I'm not responsible if someone hacks your phone.

If you want to later remove SSH, you can do so using the Remove SSH button. iNdependence will ask you if you want to also remove certain files which might be shared with other applications on your phone at that time. Please read the description of those files carefully before you choose to remove them.

SIM Unlocking

SIM unlocking allows you to use your phone with any digital cellular service provider (as opposed to being tied to one provider and a long contract).

To perform a SIM unlock, simply select the Activation / SIM tab and press the SIM Unlock Phone button. It'll take a couple of minutes, so please be patient.

Customization

The Customize tab uses a Finder-style interface and allows for drag and drop installation of ringtones, wallpapers, and applications. You can also use the Add and Delete buttons.

Third Party Applications

Please note that in order to install/remove applications, you'll need to have SSH installed on the phone as you'll be asked for the phone's IP address and root password.

To find the phone's IP address, launch the Settings application, navigate to Wi-Fi and press the blue arrow beside the name of the Wi-Fi network that your phone is connected to.

Once SSH is installed, you can install third party applications by either dragging and dropping the applications from Finder into the System/User folder, or you can press the Add button to select the application from a file chooser. To remove an application, simply select it and press the Delete button.

Ringtones

Ringtone installation is done through iTunes (it's disabled in iNdependence when using firmware 1.1.3/1.1.4).

Wallpapers

Wallpapers must be 2 PNG (.png) files (the main image and the thumbnail image). The wallpaper image sizes I use are 320x480 for the main image and 75x75 for the thumbnail image. These seem to work well for me.

One caveat when using iNdependence with firmware 1.1.3/1.1.4 is that you can only install custom wallpapers into the System folder (not the User folder). The Add and Delete buttons will stay disabled when the User wallpaper folder is selected.
